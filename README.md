# Hexo-Tipue-Search-Json

A [Hexo plugin](https://hexo.io/plugins/) to generate JSON content for [Tipue Search](http://www.tipue.com/search/docs/?d=6).

## How to install

Add plugin to Hexo:

```
npm install hexo-generator-tipue-search-json --save
```
